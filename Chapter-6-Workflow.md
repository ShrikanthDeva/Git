Workflow

Git Workflow
-

+ ``` Write Code ``` -> ``` Commit changes ``` -> ``` Make a Pull Request ```

Local Git Workflow
-

+ ``` Write Code ``` ->  ``` Stage Changes ``` ***(git add)*** -> ``` Commit changes ``` ***(git commit)*** -> ``` Push changes ``` ***(git push)***  -> ``` Make a Pull Request ```
